# One-Minute Live Demo Script

## Opening

“SMS scams are dangerous because they look urgent and local. Our tool classifies SMS messages as safe, spam, or smishing, and explains the risky signs.”

## Example 1: Safe

Paste:

> Hi mum, I will be home by 6pm. Please keep dinner for me.

Say:

“The model marks this as safe because it has no links, no credential request, and no financial pressure.”

## Example 2: Spam

Paste:

> FREE airtime for all users this weekend. Text YES to 8080 for promo details.

Say:

“This is promotional spam. It has reward language and a call to action, but it is not clearly stealing credentials.”

## Example 3: Smishing

Paste:

> Orange Money account blocked. Verify immediately: https://om-secure-login.com

Say:

“This is high risk. The explanation shows a link, urgency, account-pressure language, and financial context. The app advises the user not to click and to verify only through official channels.”

## Close

“The key value is not just prediction. It gives users a reason they can understand, which builds trust and helps prevent fraud.”
